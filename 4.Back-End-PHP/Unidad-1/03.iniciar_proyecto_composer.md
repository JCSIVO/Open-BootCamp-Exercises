En el apartado de estabilidad mínima podremos elegir entre: `stable`, `RC`, `beta`, `alpha`, `dev`

```bash
$ composer init

# ...
# ...

Minimum Stability []: stable
Package Type (e.g. library, project, metapackage, composer-plugin) []: project

Define your dependencies.

```
